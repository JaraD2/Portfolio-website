Ik ben begonnen met een schets te maken van de site in figma.

Daarna heb ik het de kleuren ingevuld. Ik heb gekozen voor een donkere achtergrond met lichte tekst. Dit is omdat ik het mooi vind en omdat het rustiger is voor de ogen.

Vervolgens ben ik begonnen met het ontwerpen van de lay-out van de website. Ik heb gebruik gemaakt van CSS Grid om de verschillende secties en elementen op de pagina te positioneren.

Nadat ik tevreden was met het ontwerp, ben ik begonnen met het implementeren van de functionaliteit. Ik heb JavaScript gebruikt om interactieve elementen toe te voegen, zoals een menu dat uitklapt wanneer erop wordt geklikt.

Tot slot heb ik de website getest op verschillende apparaten en browsers om ervoor te zorgen dat alles goed werkt en er goed uitziet op verschillende schermformaten.
